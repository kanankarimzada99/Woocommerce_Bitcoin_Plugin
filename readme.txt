Accept Bitcoin on your WooCommerce-powered website with Blockonomics.

== Description ==

Your online store must use WooCommerce platform (free wordpress plugin).
Once you installed and activated WooCommerce, you may install and activate
Bitcoin Payments for WooCommerce.
